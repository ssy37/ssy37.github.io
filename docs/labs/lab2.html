Objective

Lab 2 introduces the IMU. Through the tasks, data is collected via the IMU's accelerometer and gyroscope. Various filtering techniques are used to get clean, accurate signals. In this lab, we also received the physical robot and performed stunts with it.

Setup

AD0_VAL

The code comment reads, “On the SparkFun 9DoF IMU breakout the default is 1, and when the ADR jumper is closed the value becomes 0.” According to lecture 4 code, it is the value of the last bit on the I2C address. It is 1 right now because the AD0 pin doesn't have anything soldered to it.
